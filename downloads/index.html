What are Surfmaps?

Surf Maps are a special type of maps for the game Counter-Strike (Counter-Strike: 1.6, Counter-Strike: Source, Counter-Strike: Global Offensive) with a wide variarity of different setting, modi, gameplay. Surfmaps utilizes a engine "wonkiness" that it is possible to move / run faster on a 45+ deg slope while using the strafe keys to counter the fall and slide perpendicular to the slopes direction. Most maps don't require two Teams or even have the standard Counter-Strike weapons. Some Maps have only one gun so the first person has the opertunity to shoot the other players as a reward for being the frist to finish the maps goal. Other maps allow buying weapons. Mostly freestyle maps.

A Freestyle Counter-Strike: Global Offensive Map

Map Types

There are basicly two Map types

  • Freestyle / Tricks: Surf the Slopes as you like. There is no predefined goal. The Main characteristica is that the slopes are in an very open space and quite easy to reach.
  • Level: Like a jump and run based game these maps focus on the goal. Usualy only one way of slopes will lead to the goal. The caracteristics of such an map is (not always!) a timer to track your personal best. Also the level is more like a linear level design with death areas and high walls. Those maps are usually challangeing and hard to beat.

Server Setup

Want to setup a local Server? Read here what settings are required to play a surf map on your own. It's best and recommended to play the game with 100FPS. Otherwise the difficulty will rise because the game handles surfing based on the fps. Lower fps = less travel speed.

Counter-Strike: 1.6

Usual Server settings. (set it in the Developer Console or user the server.cfg).

  • sv_airaccelerate: 100
  • sv_wateraccelerate: 100

Counter-Strike: Source

Usual Server settings. (set it in the Developer Console or user the server.cfg). These settings will usual work. But some maps require different values or more changes like sv_gravity.

  • (required as start parameter) Tickrate: 66
  • sv_airaccelerate: 100-150
