# CS SURF

A small Windows desktop **surf** game written in C with [raylib](https://www.raylib.com/). It recreates the classic Counter-Strike "surf" movement ‚Äî Source/GoldSrc-style air strafing, ground friction, and air acceleration ‚Äî on either a built-in procedural ramp or real Half-Life / CS 1.6 `.bsp` maps, which it parses and renders itself. No external map editor, level format converter, or audio assets are required: the BSP loader, the default track, and every sound effect are generated by the game at runtime.

> made by [Erennkrb](https://github.com/Erennkrb)

---

## Features

- **Source-style surf physics** ‚Äî gravity, ground friction, acceleration, air-accelerate with a speed cap, and jumping, tuned to feel like classic CS surf movement (strafe across angled geometry to gain speed).
- **Procedural default track** ‚Äî if no maps are found, a long Catmull-Rom spline ramp is generated on launch (banked turns, side rails, floating pillars/roof pieces), so the game is playable with zero map files.
- **GoldSrc `.bsp` map loader** (`bsp.c`) ‚Äî reads Half-Life/CS 1.6 map files (BSP version 29/30) directly: parses faces, edges, and texinfo lumps, triangulates the world geometry, skips non-solid surfaces (`sky`, `clip`, `trigger`, `origin`, `hint`, `skip`, `null`), and reads the player spawn point from the entities lump (`info_player_start` / `info_player_terrorist` / `info_player_counterterrorist`).
- **Spatial hash collision** ‚Äî world triangles from a loaded BSP are bucketed into a grid for fast broad-phase collision instead of brute-force triangle checks.
- **In-game map browser** ‚Äî scans the game folder for `.bsp` files and lists them in a menu.
- **Online map downloader** (`download.c`) ‚Äî fetches a map list from `surfmaparchive.com`, downloads the chosen map as a `.zip` (via the Windows `URLDownloadToFileA` API), and extracts it with PowerShell's `Expand-Archive`. ("Online" here means *downloadable community maps*, not network multiplayer.)
- **Checkpoint timer** ‚Äî splits at evenly-spaced checkpoints along the track/map, finish time, and top speed tracking, all shown as an in-game HUD.
- **Procedural audio** ‚Äî click, jump, finish jingle, and a speed-reactive wind sound are all synthesized with simple sine/noise waveforms at startup; no `.wav`/`.mp3` files are shipped or required.
- **Custom lit shader** ‚Äî a small GLSL shader (hemisphere ambient + directional light + distance fog) is used to shade BSP world geometry.
- **Settings menu** ‚Äî resolution (3 presets), VSync, FPS cap, fullscreen, show-FPS toggle, mouse sensitivity, and FOV, persisted to `settings.cfg`.
- **Texture fallback chain** ‚Äî uses `concrete.png` if present, falls back to a source JPG texture (auto-converted to PNG by `build.bat`), and finally falls back to a procedurally generated noise texture if neither exists.

## Controls

| Input | Action |
|---|---|
| Mouse | Look |
| `W` / `A` / `S` / `D` | Move / strafe (strafe into ramp walls to surf) |
| `Space` | Jump |
| `R` | Restart run |
| `Esc` | Pause |
| `F11` | Toggle fullscreen |

## Project structure

| File | Purpose |
|---|---|
| `main.c` | Game loop, menu/state machine, player movement & physics, camera, rendering, HUD, timer/checkpoint logic |
| `bsp.c` / `bsp.h` | GoldSrc `.bsp` (v29/v30) map loader ‚Äî geometry triangulation, texture-skip filtering, spawn-point parsing |
| `settings.c` / `settings.h` | Load/save/apply game settings |
| `settings.cfg` | Saved settings (plain space-separated integers) |
| `download.c` / `download.h` | Windows-only HTTP download + zip extraction for the online map browser |
| `build.bat` | MinGW build script |
| `concrete.png` | Default world texture |
| `game.exe` | Prebuilt Windows binary |
| `raylib.dll` | raylib runtime library, bundled alongside `game.exe` |

## Requirements

- **Windows** ‚Äî the project uses `windows.h`, `urlmon.h`, and `ole32`/`urlmon` for the online map downloader, and `Expand-Archive` (PowerShell) to unzip downloaded maps. It will not build or run as-is on other platforms.
- [raylib 6.0 (win64, mingw-w64 build)](https://www.raylib.com/)
- A MinGW-w64 GCC toolchain

## Building from source

1. Install MinGW-w64 and download the raylib 6.0 win64 mingw-w64 release.
2. Open `build.bat` and update the paths at the top to match your local install:
   ```bat
   set GCC=D:\c libs\mingw64\bin\gcc.exe
   set RLINC=D:\c libs\raylib-6.0_win64_mingw-w64\include
   set RLLIB=D:\c libs\raylib-6.0_win64_mingw-w64\lib
   ```
3. Run `build.bat`. It compiles `main.c settings.c bsp.c download.c`, links against `raylib`, `opengl32`, `gdi32`, `winmm`, `ole32`, and `urlmon`, and launches `game.exe` automatically on success.

If `concrete.png` is missing but a source texture exists at `textures/Poliigon_ConcreteWorn_8690/2K/Poliigon_ConcreteWorn_8690_BaseColor.jpg`, the build script converts it to `concrete.png` for you via PowerShell before compiling.

## Running the prebuilt binary

Keep `game.exe`, `raylib.dll`, and `concrete.png` in the same folder, then run `game.exe`. Drop any `.bsp` files into that same folder to have them appear in the **Play** map list, or use **Online Maps** from the main menu to fetch one.

Optional: place a `pfp.png` in the game folder to replace the default GitHub icon shown next to the credit link on the main menu.

## Adding maps

- **Local**: copy any GoldSrc/Half-Life engine `.bsp` map (e.g. a CS 1.6 surf map) into the game folder and click **Refresh** on the map select screen.
- **Online**: open **Online Maps** from the main menu ‚Äî it lists maps scraped from `surfmaparchive.com` and downloads/extracts the selected one into a local `downloads/` folder, then loads it.

## Known limitations

- Windows-only due to the WinAPI download code and PowerShell-based unzip.
- The BSP loader reads geometry and the spawn point only ‚Äî no lightmaps, no entity logic/triggers, and the entire world is rendered with a single global texture rather than per-face textures.
- The online map browser depends on `surfmaparchive.com` staying online and keeping its current page/URL structure; if the site changes, map fetching may break.
- No license file is included ‚Äî add one before distributing or open-sourcing this further.
